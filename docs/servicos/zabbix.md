# Zabbix

O Zabbix é uma robusta plataforma de software *open-source* projetada para o monitoramento contínuo de redes, servidores, aplicações e serviços em nuvem.
* **Aplicação:** Ele atua no monitoramento proativo. Coleta e analisa métricas (como uso de CPU, tráfego de interfaces de rede e latência). Caso um serviço vital caia (como um link SIP ou um *daemon* do banco de dados), o Zabbix dispara alertas automatizados (por e-mail ou integração com mensageria) para que a equipe técnica intervenha antes que os usuários finais percebam a indisponibilidade.
