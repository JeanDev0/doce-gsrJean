# SSH (Secure Shell)


O SSH é um protocolo de rede criptografado que permite a operação segura de serviços de rede sobre uma rede não segura.
* **Aplicação:** É a ferramenta essencial para a administração remota de servidores via CLI (Command Line Interface). Diferente do antigo Telnet, o SSH garante que todos os comandos digitados e arquivos transferidos trafeguem de forma criptografada, impedindo a interceptação de senhas e dados sensíveis por *sniffers* na rede.
