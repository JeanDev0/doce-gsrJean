# LDAP 


O LDAP (Lightweight Directory Access Protocol) é um protocolo padrão de mercado utilizado para acessar e gerenciar serviços de diretório distribuídos em uma rede IP. Na prática, ele funciona como uma base de dados centralizada (uma "lista telefônica" avançada) que armazena informações de usuários, grupos e dispositivos.

* **Aplicação:** Em vez de criar credenciais separadas para cada serviço (sistema acadêmico, Wi-Fi, serviços de rede), os sistemas consultam o LDAP para validar o login do usuário, permitindo o famoso *Single Sign-On* (SSO) ou autenticação centralizada.
